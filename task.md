# AI 你画我猜 - 任务列表

## 项目概述
基于 React + TypeScript 开发的 AI 绘画识别游戏，用户在画布上绘制图形，AI 尝试识别用户绘制的内容。

## 技术栈
- 前端：React + TypeScript + Vite
- 绘图：Canvas API
- AI 模型：Deepseek AI API
- 样式：CSS
